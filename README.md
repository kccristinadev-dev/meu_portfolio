# Meu Portfolio

## Sobre
Este é meu portfolio, que reúne os projetos que desenvolvi e as habilidades que adquiri ao longo do tempo. Através deste projeto, você poderá visualizar meu trabalho e entender melhor minhas capacidades como desenvolvedor.

## Tecnologias
- **HTML**: Utilizado para estruturar o conteúdo das páginas.
- **CSS**: Utilizado para estilizar e tornar as páginas visualmente atraentes.

## Funcionalidades
- **Apresentações de Projetos**: Cada projeto possui uma apresentação detalhada com imagens e descrições.
- **Design Responsivo**: O portfolio se adapta a diferentes tamanhos de tela, garantindo uma boa experiência em dispositivos móveis e desktops.
